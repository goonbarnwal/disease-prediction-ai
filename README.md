# 🩺 Disease Prediction AI

An Intelligent Health Risk Prediction Platform that uses Machine Learning to analyze symptoms and predict possible diseases with precautions and recommendations.

## 🚀 Live Demo

Frontend:
https://disease-prediction-ai-nu.vercel.app/

Backend API:
https://disease-prediction-ai-backend.onrender.com


## ✨ Features

- AI-based disease prediction
- Analyze symptoms and get possible disease predictions
- Supports 133 symptoms
- Predicts 41 diseases
- Confidence score generation
- Disease description and precautions
- Real-time API communication
- Responsive user interface


## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Python
- Flask
- Flask-CORS

### Machine Learning
- Scikit-learn
- RandomForest Classifier
- Pandas
- NumPy

### Deployment
- Vercel (Frontend)
- Render (Backend)


## ⚙️ How It Works

1. User enters symptoms.
2. Frontend sends symptoms to Flask API.
3. Machine Learning model analyzes symptoms.
4. System returns predicted disease with confidence and precautions.
